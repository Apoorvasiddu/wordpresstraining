== White Label Theme ==

To enable white-label settings, you need to define two constants in the config.php file. Replace YourSlug with your desired name. 
Add this code before the this line: /* Add any custom values between this line and the "stop editing" line. */

define( 'WOFFICE_WHITELABEL', true );
define( 'WOFFICE_WHITELABEL_NAME', 'YourSlug' );
